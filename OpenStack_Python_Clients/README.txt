Steps to excute "OpenStack_Python_Clients" package

1. Execute "OpenStack_Python_Clients" package in CLI as below,
#python OpenStack_Python_Clients

2. It asks for OpenStack login credentials, you need to specify your Keystone username, password, tenant name, and Server IP.

3. Creation of router, network, add network interface to a router using neutronclient

4. Then it invokes instance_creation() to launch VM on to a target network.

5. Once VM launched on VM, it displays the actual status of provisioned VM.

6. Assign a floating-ip to a provisioned VM
